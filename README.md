An enhanced version of the Hollow Knight web port fork by HorizonOfficia https://github.com/HorizonOfficia/hollow-knight. This enhanced version adds the following:
- FPS counter accessible by pressing ALT+,
- Render scale slider accessible by pressing ALT+. (reducing the render scale can help with performance, especially on Chromebooks)
- Save manager that allows save files to be imported and exported accessible by pressing ALT+/
- Improved initial loading screen that shows relevant information

Use hk-web-enhanced.html for these enhancements.
