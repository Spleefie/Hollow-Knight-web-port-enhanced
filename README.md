An enhanced version of the Hollow Knight web port fork by HorizonOfficia https://github.com/HorizonOfficia/hollow-knight. This enhanced version adds the following:
- A FPS counter accessible by pressing ALT+,
- A render scale slider accessible by pressing ALT+. (reducing the render scale can greatly help with performance, especially on Chromebooks)
- A save manager that allows save files to be imported and exported accessible by pressing ALT+/
