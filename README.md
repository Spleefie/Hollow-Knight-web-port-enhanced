An enhanced version of the Hollow Knight web port fork by HorizonOfficia https://github.com/HorizonOfficia/hollow-knight. This enhanced version adds the following:
- A FPS counter accessable by pressing ALT+,
- A render scale slider accessable by pressing ALT+. (reducing the render scale can greatly help with performance, especially on Chromebooks)
- A save manager that allows save files to be imported and exported accessable by pressing ALT+/
